# Monitor to Comport Association File for Elo Touchscreens.
# File to link serial touchscreens with available video monitors.
# Update this file and restart the computer to take effect.
#
#  *** IMPORTANT: All video screens have to be listed in this file. *** 
#
# Association record syntax is shown below.
# Monitor Number, Comport Name
#
# Example association records are shown below:
#
# Monitor[Video Screen] 0 associated with Touchscreen on Comport 0 [ttyS0]
# 0, ttyS0
#
# Monitor[Video Screen] 1 associated with no Touchscreen
# 1, N
#
0, ttyS0 
1, ttyS1 
#
# End of File
