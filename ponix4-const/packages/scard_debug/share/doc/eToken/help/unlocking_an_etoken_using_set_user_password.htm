Unlocking an eToken using Set User Password

To unlock a token using Set User Password:

  1. Log on to the eToken as an administrator.

  2. Do one of the following:

  3. Click the Set User Password icon:

  4. Right-click the eToken in the left pane and select Set User Password from the shortcut menu.

The Set eToken Password window opens.

  1. Enter a new password in the New Password and Confirm Password fields.

The new password must meet password quality settings as defined for the eToken.

  1. Set the Set maximum number of logon failures to the required number.

  2. Click OK.

The eToken is unlocked.

You can now log on as a user with the new password.
