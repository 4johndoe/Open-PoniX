Launching the eToken NG-Flash Partition Application

Note: This feature is available only in Windows

 

To launch eToken NG-Flash Partition Application:

  1. Open eToken PKI Client Settings (See Opening eToken PKI Client Settings).

  2. Right Click on eToken PKI Client Settings in the left pane.

  3. Select Launch eToken NG-Flash Partition Application from the menu.

The eToken NG-Flash Partition Application opens.

  1. Insert an eToken NG-Flash.

The application displays information about the inserted eToken NG-FLASH.

  1. Set the eToken parameters as required and click Run Partition.

The eToken NG-Flash is reconfigured.
