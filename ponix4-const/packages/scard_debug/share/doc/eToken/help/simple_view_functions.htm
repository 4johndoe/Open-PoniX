Simple View Functions

In the right pane, you can select any of the enabled buttons to perform the action described.

 

Function

Button

Rename eToken - sets the eToken name

Change Password – changes the eToken user password

Unlock eToken – resets the user password via a challenge response mechanism .Enabled only when an administrator password has been initialized on the eToken.

Clear eToken - removes deletable data from the eToken

View eToken Info – provides detailed information about the token

Disconnect eToken Virtual – disconnects the eToken Virtual, with an option for deleting it
